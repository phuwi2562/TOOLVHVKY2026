# คู่มือออนไลน์ อสม.ตำบลคำใหญ่ NCDs 2569

เปิดใช้งานจากไฟล์ `index.html` ได้ทันที หรือเปิดผ่านเว็บเซิร์ฟเวอร์ในเครื่องเพื่อทดสอบเหมือนเว็บจริง

```powershell
python -m http.server 8765
```

แล้วเปิด `http://localhost:8765`

## ไฟล์สำคัญ

- `index.html` หน้าเว็บคู่มือบนมือถือ
- `styles.css` รูปแบบหน้าจอและ responsive layout
- `script.js` ค้นหา เช็กลิสต์ แผนกิจกรรม และตัวเปิดดูหน้าคู่มือต้นฉบับ
- `assets/pages/` ภาพหน้าคู่มือต้นฉบับ 81 หน้า

เนื้อหาออกแบบเป็นคู่มือใช้งานเร็วสำหรับ อสม. ได้แก่ ขั้นตอนปฏิบัติงาน 5 บทบาท การคัดกรอง NCDs การให้คำแนะนำปรับพฤติกรรม เช็กลิสต์ลงพื้นที่ แบบออกแบบกิจกรรมชุมชน และคลังหน้าต้นฉบับ
